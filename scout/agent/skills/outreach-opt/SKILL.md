# Ниша: Опт и дистрибуция

## Кому пишем

Оптовые базы, дистрибьюторы, поставщики оборудования, продуктов питания, стройматериалов.

## Jobs to Be Done

- Партнёры сами видят остатки и делают повторный заказ
- Убрать ручной ввод из WhatsApp/Excel в 1С
- Прайсы и скидки для разных категорий клиентов
- Сайт как витрина + вход в личный кабинет

## Типовые боли

- «Менеджеры тонут в переписке с дилерами»
- «Прайс в PDF на почте — устарел ещё вчера»
- «Новый клиент не понимает ассортимент с сайта»
- «Заявки теряются между сайтом и мессенджером»

## Что предлагать

| Сигнал | Услуга |
|--------|--------|

## Фреймворк по умолчанию

**BAB** — сейчас (хаос в заказах) → как могло бы быть (кабинет) → разбор.

## Зацепки

- «Ассортимент серьёзный» из отзывов
- Упоминание дилеров/опта на сайте
- Несколько телефонов/отделов — хаос в заявках
