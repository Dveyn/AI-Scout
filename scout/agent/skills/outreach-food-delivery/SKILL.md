# Ниша: Доставка еды

## Кому пишем

Сети и локальные бренды доставки: суши-бары, пиццерии, wok, бургерные, паназиатская кухня.
Региональные игроки рядом с федеральными сетями (Додо, Суши Даром, Майбокс и т.п.).

## Jobs to Be Done

- Свой сайт с онлайн-заказом — меньше комиссии агрегаторам
- Меню, акции, комбо — обновляются без разработчика
- Повторные заказы: бот, Mini App, сохранённые адреса
- SEO: «доставка суши [город]», «пицца на дом [район]»
- Мобильный заказ за 30 секунд — основной трафик с телефона

## Типовые боли

- «80% заказов через Яндекс Еда — комиссия 25–35%»
- «Сайт есть, но заказать нельзя — только звонок или агрегатор»
- «Меню на сайте устарело, акции не видны»
- «Конкурент в поиске выглядит современнее»
- «Нет своей базы клиентов — только телефон в агрегаторе»

## Что предлагать

| Сигнал | Услуга |
|--------|--------|
| Нет онлайн-заказа на сайте | Сайт с корзиной и оплатой/предзаказом |
| Только агрегаторы в отзывах | Свой канал заказа + SEO |
| Слабая мобилка | Адаптив + быстрая форма заказа |
| Несколько точек | Единое меню, выбор филиала |
| Повторные клиенты | Telegram-бот / Mini App |

## Зацепки

- Рейтинг на картах высокий, сайт слабый — контраст
- В отзывах «удобно заказать в приложении» у конкурента
- Много филиалов на картах — сигнал сети
- Упоминание iiko/R-Keeper на сайте — готовы к интеграции

## CTA

«Покажу на примере вашего сайта, где теряются заказы в пользу агрегаторов и звонка — 30 минут, без обязательств.»

## Не предлагать

B2B-портал с 1С суши-бару. Сложный маркетплейс одной точке без сети.
