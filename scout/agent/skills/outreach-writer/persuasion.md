# Persuasion Layer (из cold-email, offers, marketing-psychology)

Синтез глобальных skills для **первого касания** Scout. Цель — reply, не «продать в лоб».

## Выбери фреймворк под ситуацию

| Ситуация | Фреймворк | Структура |
|----------|-----------|-----------|
| Есть проблемы с сайта в аудите | **PAS** | Проблема → что теряют → мягкое решение + CTA |
| B2B/опт, сайт норм, хаос в заказах | **BAB** | Сейчас (Excel/WhatsApp) → как могло бы быть (портал) → мост (разбор) |
| Мессенджер, мало данных | **Mouse Trap** | 1 наблюдение + бинарный вопрос («актуально?» / «интересно глянуть?») |
| Сильные отзывы на картах | **PPP** | Искренний комплимент по отзыву → картина улучшения → лёгкий push |
| Технический ЛПР редко | **QVC** | Вопрос про боль → ценность одной фразой → низкий порог CTA |

**Правило:** персонализация должна вести к проблеме. Убери первое предложение — если письмо всё ещё логично, перепиши зацепку.

## Value Equation (оффер «бесплатный разбор»)

```
Ценность = (Желаемый результат × Вероятность что получится)
           ÷ (Время до результата × Усилия с их стороны)
```

Как усилить **без вранья**:

| Рычаг | В письме |
|-------|----------|
| Результат ↑ | «больше заявок с сайта» / «меньше ручного ввода в 1С» — их словами |
| Вероятность ↑ | «на примере вашего сайта», «покажу конкретно у вас» |
| Время ↓ | «30 минут», «без подготовки» |
| Усилия ↓ | «без обязательств», «просто ответьте удобное время» |

## Психология (этично, для cold outreach)

1. **Loss aversion** — сильнее «получите», чем «можете получить».  
   ✅ «Часть заявок с телефона уходит, пока сайт грузится»  
   ❌ «Мы улучшим ваш сайт»

2. **Reciprocity** — сначала ценность (наблюдение + insight), потом просьба.  
   Разбор бесплатный = подарок, не «запишитесь на продажу».

3. **Zero-price effect** — «бесплатно» и «без обязательств» снимают барьер сильнее скидки.

4. **Status-quo bias** — не атакуй «у вас всё плохо».  
   ✅ «Когда начнёте масштабировать заказы, Excel обычно первым ломается»

5. **Jobs to Be Done** — не «сделаем сайт», а «чтобы заявки не терялись между телефоном и менеджером».

6. **Contrast** — до/после в одном предложении, без фантазий.

## Subject lines (cold-email data)

- 2–5 слов, строчные, без капса и эмодзи
- Выглядят как письмо коллеги, не реклама
- Про **их** бизнес или упущенную выгоду
- Имя компании в subject — ок, если естественно

Примеры: `металлпром — мобилка`, `заявки с сайта`, `по сайту оптторг`

## CTA — низкий порог (не «звонок на 30 мин» в лоб)

Сильные:
- «Интересно глянуть на примере вашего сайта?»
- «Стоит 30 минут разбора — или пока не актуально?»
- «Напишите удобное время — покажу без обязательств»

Слабые:
- «Свяжитесь с нами»
- «Ждём вашего ответа»
- «Запишитесь на демо»

## Quality gate (перед submit_lead_result)

Ответь «да» на все пункты — иначе перепиши message:

1. Первое предложение — только про **них** (факт с сайта/карт/отзыва)?
2. Есть **одна** бизнес-боль в деньгах/заявках/времени?
3. «Вы/ваш» чаще, чем «мы/наш»?
4. Один CTA, ответ одной строкой?
5. Прочитал вслух — звучит как человек, не рассылка?
6. **Ты бы ответил**, если бы получил это письмо?
7. Нет слов из бан-листа product-marketing?

## hook, pains, product_angle

- **hook** = первое предложение message (дословно)
- **pains** = 2–3 формулировки потерь бизнеса (не технические issue)
- **product_angle** = одна фраза «для них релевантно X, потому что Y»
