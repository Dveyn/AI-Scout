# Ниша: Производство

## Кому пишем

Заводы, цеха, металлообработка, пищевое производство, мебельное производство, упаковка.

## Jobs to Be Done

- B2B-заказы с сайта без потери на мобилке
- Каталог продукции + заявка/прайс для оптовиков
- Связка с 1С: остатки, статусы заказов
- Доверие: сертификаты, объёмы, «мы производитель»

## Типовые боли (говори их словами)

- «Ищут поставщика с телефона — сайт не открывается»
- «Заявки в WhatsApp, менеджер переписывает в Excel»
- «Конкурент в выдаче Яндекса выглядит современнее»
- «Нет нормального каталога продукции на сайте»

## Что предлагать ВебШтрих

| Сигнал | Услуга |
|--------|--------|
| Слабый сайт, нет HTTPS/мобилки | Корпоративный сайт под B2B-заявки |
| Много SKU, опт | Каталог + личный кабинет |
| Excel + 1С | B2B-портал с интеграцией 1С |
| Есть только карточка на картах | Сайт + форма заявки |

## Зацепки с карт/сайта

- Название цеха/продукции («металлоконструкции», «мебель на заказ»)
- Отзывы про качество или сроки
- Упоминание опта/дилеров на сайте
