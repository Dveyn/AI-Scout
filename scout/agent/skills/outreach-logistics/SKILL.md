# Ниша: Логистика и перевозки

## Кому пишем

Грузоперевозки, складская логистика, курьерские службы, экспедирование, ТК.

## Jobs to Be Done

- Заявка на расчёт перевозки с сайта за 30 секунд
- Статус груза для клиента без звонка менеджеру
- Доверие: лицензии, парк, география
- Мобильные заявки — водители и диспетчеры ищут с телефона

## Типовые боли

- «Клиент не дозванивается» (часто в отзывах на картах)
- «Нет формы расчёта — уходят к конкуренту»
- «Сайт 2015 года — не внушает доверие для B2B»
- «Заявки в Telegram, нет учёта»

## Что предлагать

| Сигнал | Услуга |
|--------|--------|
| Нет формы / калькулятора | Сайт с заявкой и калькулятором |
| Отзывы «долго отвечают» | Бот + CRM/уведомления |
| Слабая мобилка | Сайт под мобильный трафик |
| Много направлений | Структура услуг + SEO |

## Фреймворк

**Mouse Trap** для мессенджера + **PAS** если слабый сайт в аудите.

## Зацепки

- Отзыв про скорость ответа или срыв сроков
- Перечень направлений/услуг на сайте
- Рейтинг на картах vs слабый сайт
